Interactive Chart API Demo

Presented by MOD/Labs, the Interactive Chart API is useful for determining the historical value of a company's stock. The response contains end-of-day historical price information for companies traded on the BATS Exchange in the United States. The result set includes the name and symbol for the company, as well as a dataset containing open-high-low-close (OHLC) values, labeling helpers, coordinates and more.

Read the Interactive Chart API documentation or check out our other v2 APIs.
